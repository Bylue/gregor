Frage: Worum handelt das Gedankenexperiment "Schrödingers Katze"?

Allgemeine Inforamtionen

  • Wann 1935
  • Von: Erwin Schröder
  • Warum: um einen Schwachpunkt der Kopenhagener Interpretation
    der Quantenmechanik in Bezug auf die physikalische Realität aufzuzeigen
Das Experiment: Es befindet sich eine Katze in einer Geschlossenen Box.
Man steckt sie zusammen in die Box mit einem Messglas, wo sich einen minimale Dosis Uran befindet.
So viel, dass es innerhalb einer Stunde entweder zerbricht oder ganz bleibt.
Wenn es zerbricht, setzt es eine Dosis Blausäure frei, welche die Katze tötet.
Wenn man nun nicht die Box öffnet, dann weiss man nicht, ob die Katze tot oder lebendig ist.
Das Gedankenexperiment beruht darauf, dass wenn ein system zwei verschiedene Zustände annehmen kann, dann muss es auch
die Kohärente Überlagerund ein möglicher Zustand der Zwei zustände sein.

Das würde der Theorie der Kopenhagener Interpretation wiedersprechen.
Denn durch das Gesetz der Quantenmechanik würde die Karte in einen
zustand zwischen Leben und Tod versetzt werden.
